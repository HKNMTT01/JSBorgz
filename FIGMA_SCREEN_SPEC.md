# Figma screen specification

Use the existing blue/green JETAMA corporate direction and keep the first release functional rather than over-designed.

## Desktop frames

1. Login — corporate dam image, logo, email/password, support link.
2. Employee dashboard — leave balance, claims, notices, policies and quick actions.
3. HR dashboard — employee totals, pending approvals, recent leave and claims.
4. Employees — searchable table, filters, add/edit drawer.
5. Leave application — employee summary, date range, calculated days, recommender, approver, private attachment.
6. Leave approval — request details, balance, document preview, approve/reject and comment.
7. Claim submission — category, amount, description, receipt.
8. Claim approval — HR, Finance and CEO stage indicator.
9. Notices — list and publish form.
10. Policies — versioned document list and secure viewer.
11. Profile and security — profile details and password change.
12. Audit log — administrator-only activity table.

## Mobile frames

Prioritize dashboard, leave application, claims, notices and approval actions. Convert tables into stacked rows below 768px.

## Components

- Top navigation and responsive sidebar
- KPI tile
- Status badge
- Data table / mobile row
- Empty state
- Confirmation modal
- File uploader
- Approval timeline
- Toast notification
- Form field and validation state
